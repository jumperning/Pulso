Banners del Hero

Imágenes y textos que aparecen en la portada del menú público.

image_not_supported

Todavía no hay banners

Creá el primero con el botón de arriba

Tabla Supabase requerida

create table banners (
  id uuid primary key default gen_random_uuid(),
  business_id uuid,
  title text,
  subtitle text,
  image_url text,
  tag_label text,
  btn_primary_label text,
  btn_secondary_label text,
  active boolean default true,
  display_order integer default 0,
  created_at timestamptz default now()
);
create policy "admin_banners" on banners for all using (true);

Categorías visibles

Controlá qué categorías se muestran y en qué orden en el menú.

Categorías detectadas en productos

Activá o desactivá cada categoría. Las desactivadas no aparecerán en el menú público.

info

El orden de las categorías en el menú es alfabético por defecto. Para cambiar el orden, modificá los nombres de las categorías en los productos desde el panel de Inventario.

Textos y Marca

Personalizá los textos que aparecen en el menú público.

storefront Marca

share Redes Sociales

segment Footer

warning

Estos textos se guardan en la tabla menu_settings de Supabase. El menú los leerá en cada carga. Ver SQL requerido abajo.

Tabla Supabase requerida

create table menu_settings (
  id uuid primary key default gen_random_uuid(),
  business_id uuid unique,
  brand_name text default 'ONCEYDOCE',
  tagline text,
  newsletter_title text default 'JOIN THE CULT.',
  instagram_url text,
  tiktok_url text,
  twitter_url text,
  copyright_text text,
  points_rate integer default 100,
  updated_at timestamptz default now()
);
create policy "admin_settings" on menu_settings for all using (true);

Carrito y Pedidos Online

Controlá si los usuarios pueden hacer pedidos desde el menú público.

Carrito activo

Cuando está desactivado, los usuarios pueden ver el menú pero no pueden agregar productos al carrito ni hacer pedidos.

check_circle Carrito habilitado — los usuarios pueden pedir

Sistema de puntos

Activá o desactivá el sistema de fidelización. Los usuarios no verán su saldo de puntos si está desactivado.

Requerir login para pedir

Si está activo, el usuario debe estar registrado e identificado para poder confirmar un pedido.

Mensaje cuando el carrito está cerrado

Tabla Supabase requerida (extender menu_settings)

alter table menu_settings
  add column if not exists cart_enabled boolean default true,
  add column if not exists points_enabled boolean default true,
  add column if not exists login_required boolean default false,
  add column if not exists cart_closed_msg text;

OnceyDoce Clientes

Tabla Supabase requerida

create table customer_profiles (
  id uuid primary key references auth.users(id),
  business_id uuid,
  full_name text,
  email text,
  points integer default 0,
  total_orders integer default 0,
  total_spent numeric default 0,
  created_at timestamptz default now(),
  updated_at timestamptz default now()
);
-- Política para que el admin pueda leer todos:
create policy "admin_read_customers" on customer_profiles
  for select using (true);
create policy "users_manage_own" on customer_profiles
  for all using (auth.uid() = id);
